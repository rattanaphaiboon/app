      ข้อมูลพนักงานดึงจาก Users Sheet · กดรีเฟรชเมื่อ HR แก้ไข Sheet
      📖 วิธีตั้งค่า Google OAuth (5 นาที)

      1. ไปที่ Google Cloud Console → Credentials

      2. เลือกโปรเจคหรือสร้างใหม่ (เช่น "Rattana Apps")

      3. ถ้ายังไม่มี OAuth consent screen → กดสร้าง User Type = Internal (สำหรับ Google Workspace) หรือ External

      4. Credentials → "+ CREATE CREDENTIALS" → "OAuth client ID"

      5. Application type = Web application

      6. Authorized JavaScript origins → เพิ่ม URL ที่ใช้ deploy เช่น
      https://suratpoom.github.io

      7. CREATE → Copy Client ID มาวางในช่องด้านบน

      8. เปิด Users Sheet → Share → "Anyone with the link can view"
