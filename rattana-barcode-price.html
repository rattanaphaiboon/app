เกี่ยวกับแอป

แอปนี้ใช้ค้นหาชื่อสินค้าจากบาร์โค้ดสากล (EAN/UPC) ตรวจสต็อกในระบบรัตน และเปิดลิงก์ค้นหาราคาในร้านค้าออนไลน์ไทย

แหล่งข้อมูลสินค้า

  • Open Food Facts — อาหาร เครื่องดื่ม FMCG ทั่วโลก
  • UPC Item DB — สินค้าสากล (ฟรี trial)
  • Google Sheets รัตน — สต็อก+ราคาภายใน

สแกนเนอร์ V2

ใช้ html5-qrcode + BarcodeDetector API ประสิทธิภาพสูง รองรับ Android/iOS กรอบทองพร้อมไฟฉาย

หมายเหตุ

  • สินค้าไทยบางตัวอาจไม่อยู่ในฐานข้อมูลสากล
  • ราคา Lazada/Shopee เป็น B2C ไม่ใช่ราคาส่งรัตน
  • ประวัติเก็บใน browser นี้เท่านั้น

🤖 Claude API Key

ใส่ key เพื่อดึงราคาตลาดไทยอัตโนมัติ (BigC, Lotus, Makro, Lazada, Shopee, Tops)

Key เก็บใน browser เครื่องนี้เท่านั้น ไม่ส่งออกไปไหน

📝 Apps Script URL (ขอเปลี่ยนราคา)

ใส่ URL ของ Google Apps Script เพื่อส่งคำขอเปลี่ยนราคาเข้า Sheet โดยตรง

วิธีสร้าง Apps Script →
1. เปิด Google Sheet ที่ต้องการ → Extensions → Apps Script
2. วางโค้ดนี้แทนที่ทั้งหมด:
function doPost(e){
  const ss=SpreadsheetApp.openById(
    '1_zwgBL77CNQVgQW_jEvwVJ8Yo8FAeGfWpqlyLZlnooE');
  const sh=ss.getSheets().find(
    s=>s.getSheetId()==300494340)||ss.getSheets()[0];
  const d=JSON.parse(e.postData.contents);
  sh.appendRow([d.date,d.time,d.requester,d.role,
    d.codeEA,d.codeCS,d.name,
    d.eaOld,d.eaNew,d.paOld,d.paNew,
    d.bpOld,d.bpNew,d.cs03Old,d.cs03New,
    d.cs02Old,d.cs02New,d.cs01Old,d.cs01New,
    d.cs00Old,d.cs00New,d.reason,d.status]);
  return ContentService
    .createTextOutput(JSON.stringify({ok:true}))
    .setMimeType(ContentService.MimeType.JSON);
}
3. Deploy → New deployment → Web app → Execute as: Me → Who has access: Anyone
4. คัดลอก URL มาวางข้างบน
